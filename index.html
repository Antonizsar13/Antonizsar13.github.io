Игра Сапер

    Повтор игры

      Справка

      Доступные функции:
      - Новая игра: Позволяет начать новую игру с заданными параметрами (ширина, высота и количество мин).
      - Просмотр прошлых игр: Открывает список всех сохранённых игр. Чтобы посмотреть реплей конкретной игры, нажмите на кнопку "Просмотр прошлых игр" и выберите интересующую игру из списка.
      - Повтор игры (Replay): Показывает пошаговую историю выбранной игры, включая информацию о каждом ходе (координаты и результат).
      - Очистка базы данных: Удаляет все сохранённые игры из базы данных.

      Как играть:
      1. Введите ваше имя в поле "Имя игрока".
      2. Установите параметры игры (размер поля и количество мин).
      3. Нажмите кнопку "Новая игра", чтобы начать новую игру.
      4. Щёлкайте по ячейкам на игровом поле, чтобы открыть их. Если вы откроете ячейку с миной, игра закончится.
      5. Цель игры — открыть все ячейки без мин.
